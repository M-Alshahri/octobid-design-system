LOGO CHARACTER
The Octobid logomark is an abstract octopus constructed entirely from geometric arches and vertical bars — evoking both the brand name and the tentacle-like reach of a platform connecting many hospitals to many suppliers. The mark is line-art only, using a consistent stroke weight, with no fills.

Usage Rules

✓ Do
Use #2547A1 as the only primary action color. Reserve it for CTAs, links, and the brand mark.
✗ Don't
Use multiple competing accent colors. Octobid's palette is intentionally monochromatic — one dominant blue.
✓ Do
Use full-width primary buttons for the single main action per screen (Sign In, Next, Submit Request).
✗ Don't
Place two primary blue buttons side-by-side. Use primary + secondary pairing for dual-action scenarios.
✓ Do
Apply #EDF2FB tint to input backgrounds to distinguish them visually from the white card surface.
✗ Don't
Use pure white inputs on a white card — inputs need the tint to be distinguishable.
✓ Do
Use the logo mark on solid brand blue (#2547A1) in white, or on white/light surfaces in brand blue.
✗ Don't
Place the logo on busy photographic backgrounds or use the logo in non-brand colors.
